# DOC_ID: kbm_definition
# CATEGORY: kbm
# TITLE: Что такое КБМ

КБМ — коэффициент бонус-малус.

Используется при расчете стоимости ОСАГО.

Минимальное значение КБМ: 0.46

Максимальное значение КБМ: 3.92

КБМ закрепляется за водителем, а не за автомобилем.

---

# DOC_ID: kbm_recalculation
# CATEGORY: kbm
# TITLE: Пересчет КБМ

Пересчет КБМ выполняется ежегодно 1 апреля.

Если водитель не становится виновником ДТП, КБМ снижается.

Если водитель становится виновником ДТП, КБМ увеличивается.

---

# DOC_ID: kbm_check
# CATEGORY: kbm
# TITLE: Проверка КБМ

Проверка осуществляется через НСИС.

Для проверки требуются данные водительского удостоверения.

При обнаружении ошибки можно подать обращение через НСИС.
